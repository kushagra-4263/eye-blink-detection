# 👁️ Eye Blink Detection System

## 📌 Overview

This project detects eye blinks in real-time using computer vision and facial landmark detection.

## ⚙️ Tech Stack

* Python
* OpenCV
* MediaPipe

## 🚀 Features

* Real-time face detection
* Eye blink detection using EAR
* Lightweight and fast

## ▶️ How to Run

1. Install dependencies

```
pip install -r requirements.txt
```

2. Run the project

```
python main.py
```
